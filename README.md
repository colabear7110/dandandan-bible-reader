# 단.단.단 성경통독 앱

교회 유튜브 재생목록을 연결해서 매일 성경통독 영상을 바로 볼 수 있는 정적 웹앱입니다. 2026년 성경읽기표 일정과 `단.단.단 성경통독` 재생목록이 기본으로 연결되어 있습니다.

## 사용 방법

1. `index.html`을 브라우저로 엽니다.
2. 오른쪽 위 설정 버튼을 누릅니다.
3. 다른 재생목록을 쓰고 싶다면 교회 유튜브 재생목록 URL을 붙여 넣고 저장합니다.

진행률과 완료 표시는 현재 브라우저에 저장됩니다.

## 자동 영상 갱신

Google Cloud API 키 없이 YouTube 재생목록 RSS를 읽어 `videos.js`를 갱신합니다.

```bash
node scripts/update-videos.mjs
```

GitHub에 올리면 `.github/workflows/update-videos.yml`이 매일 오전 7시 1분(KST)에 실행되어 새 영상을 자동으로 추가합니다. 수동 실행은 GitHub Actions의 `Update YouTube Videos` 워크플로에서 `Run workflow`를 누르면 됩니다.

RSS는 최근 항목 중심이라 자동 실행이 오래 멈추면 일부 날짜를 놓칠 수 있습니다. 매일 실행되도록 유지하는 것이 좋습니다.
